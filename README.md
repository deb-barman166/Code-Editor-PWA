🚀 Offline Code Editor (VS Code Style)

A powerful, mobile-friendly offline code editor web app inspired by modern development tools like VS Code. This project is designed to run directly in the browser and works even without an internet connection.

---

📌 Overview

This is a Progressive Web App (PWA) that allows users to write, edit, upload, and download code files directly from their mobile or desktop browser. It is lightweight, fast, and optimized for offline usage.

---

✨ Features

- 📝 Code editing with syntax highlighting (HTML, CSS, JavaScript, Python, C++, JSON)
- 📂 Upload and edit existing files
- 💾 Download edited files
- 🆕 Create, rename, and delete files
- 🗂️ Multi-tab file system
- 🔍 Search inside files
- 🌙 Dark / Light mode toggle
- 📱 Fully mobile responsive
- 📴 Works offline (PWA enabled)
- ⚡ Fast and lightweight performance

---

🧠 Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Monaco Editor (VS Code editor engine)
- Service Worker (for offline support)
- IndexedDB / LocalStorage

📴 Offline Support

This app uses Service Worker caching, allowing it to:

- Load instantly after first visit
- Work without internet connection
- Store files locally in browser storage
